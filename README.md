# 2hu-demo
SDL2 demo based on Touhou Project

### Usage:
```
git clone https://github.com/feydor/2hu-demo
cd 2hu-demo
meson build
ninja -C build
./build/2hu-demo
```

Run the unit tests:
```
cd build
meson test
```
### Dependencies:
meson sdl2 sdl2_image sdl2_mixer

### Controls:
z: fire
wasd: movement
